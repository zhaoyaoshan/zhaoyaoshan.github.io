Hexo自定义域名

作者 木 日期 2018-12-16
HEXO
Hexo自定义域名

报错提示Domain dose not resolve to the Github Pages server

方案1(域名到域名)

域名解析时，只添加CNAME解析，指向域名xxx.github.io

添加@解析和www解析，其它可按情况添加。（泛解析亦适用）

方案2（域名到ipv4）

在Linux（Debian）命令行下输入

dig xxx.github.io

在’;;ANSWER SECTION:’下会显示被分配到的ip地址

将这些地址如数添加解析

dig github.com

Github的地址也同上添加解析

_config.yml 配置应修改如下：

url:自定义域名

skip_render: CNAME

且在source目录下拥有文件CNAME存在，文件格式为所有文件
