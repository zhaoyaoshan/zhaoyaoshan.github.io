HEXO Linux
Hexo部署

系统：Debian

有关root权限等需自行考虑

apt-get install nodejs
apt-get install npm
npm install hexo-cli -g
mkdir blog
cd blog
hexo init
hexo s

打开浏览器登录http://localhost:4000 即可见到网站。
